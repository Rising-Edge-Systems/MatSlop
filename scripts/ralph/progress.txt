## Codebase Patterns
- download-octave.js: per-platform pinned versions (WIN_OCTAVE_VERSION, MAC_OCTAVE_VERSION). Windows ships from ftp.gnu.org/gnu/octave/windows (NOT ftpmirror.gnu.org — that redirector 502s). macOS is pinned to octave-app v9.2 (release ceiling). Linux is intentionally NOT bundled — no maintained AppImage source; the script exits cleanly with package-manager guidance and src/main/octaveConfig.ts handles PATH lookup at runtime.
- Tests that assert download-octave.js content live at tests/unit/download-octave-{linux,mac}.test.ts and read the script as a string — any URL or constant change needs matching regex updates there.
- octaveConfig.ts retains its legacy squashfs-root branch as a fallback for users who manually drop an AppImage into resources/octave/, even though the download script no longer populates it.
